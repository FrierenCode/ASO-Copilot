## Overview
- A pnpm monorepo running `apps/web` (Next.js) and `apps/api` (Cloudflare Worker).
- `packages/shared` keeps request/response schemas shared between API and web.
- `packages/scoring` contains copy scoring and recommendation logic.
- `POST /generate` returns generated variants with score and recommendations.
- `GET /v1/entitlements` returns current plan and monthly usage status.
- `POST /webhooks/polar` syncs entitlements from billing events.
- Data model changes are managed with 1 D1 migration file(s).
- Documentation assets: `docs/` 10 Markdown file(s), `docs_result/` 4 Markdown file(s).
