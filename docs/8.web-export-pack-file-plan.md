1. 수정: [apps/web/src/app/page.tsx](/C:/Users/Administrator/aso-copilot/apps/web/src/app/page.tsx)  
결과(`result`) 섹션 하단에 `Download JSON`, `Download CSV` 버튼 2개를 추가한다. 버튼은 `result`가 있을 때만 보이도록 기존 조건부 렌더링 블록 내부에 배치한다.

2. JSON 내보내기 구현(동일 파일)  
`Blob([JSON.stringify(result, null, 2)], { type: 'application/json;charset=utf-8' })`로 JSON Blob을 생성하고 다운로드 처리한다. 파일명은 `aso-copilot-export.json`으로 고정한다.

3. CSV 내보내기 구현(동일 파일)  
외부 라이브러리 없이 CSV 문자열을 생성한다. 단일 행(flatten) 기준으로 아래 값을 포함한다.  
- `score`  
- `breakdown`의 각 필드  
- `recommendations`: 추천 문구 배열을 `|`로 join (현재 스키마 필드명 `recommendation`)  
- `variants`: `A`, `B`, `C` 각각을 `|`로 join  
생성한 CSV를 Blob으로 내려받고 파일명은 `aso-copilot-export.csv`로 고정한다.

4. 제약 조건 준수 확인(동일 파일)  
추가 패키지/외부 라이브러리는 사용하지 않는다. 브라우저 기본 API(`Blob`, `URL.createObjectURL`, 임시 `<a download>`)만 사용한다.
